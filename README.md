# E-commerce-Responsive-Webpage-
A fully responsive e-commerce website front-end developed with HTML5 and CSS3. The project focuses on clean UI design, responsive layouts, and an enhanced user experience across desktop, tablet, and mobile devices.
This project is a responsive e-commerce web page created using HTML and CSS. It includes a modern homepage design, product sections, navigation bar, and responsive styling to ensure compatibility across different screen sizes. The purpose of this project is to practice front-end web development and responsive design principles
